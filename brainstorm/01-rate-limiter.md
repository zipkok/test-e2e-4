# Rate Limiter

## 배경

API 요청을 제한하는 기능이 필요한데, 정확히 어떤 방식이 좋을지 모르겠다.

## 아이디어 (애매한 상태)

- 어떤 알고리즘? Token bucket? Sliding window? Fixed window?
- 시간 단위는? 초당? 분당?
- 초과 시 에러? 대기? 드롭?
- 키 기반? (IP별, 사용자별?) 아니면 글로벌?
- 분산 환경 고려? 아니면 단일 프로세스?
